# 셔틀 예약 시스템 developed by anon

# 요구 조건 : Windows10, Pycharm with anaconda2, chromedriver
	* anaconda3에서 테스트는 된 적 없음
	* chormedrive는 46번째 줄에 있는 것과 같이 주소를 부여하면 됨. 인터넷에서 다운로드 받으셈.

# 더 많은 조건을 넣지 않기 위해 셔틀 예약이 풀리는 시간보다 23시간 59분 적게 남았을 때 부터만 사용 가능함 (예시 : 2019년 3월 10일 오후 2시에 예약을 걸고 싶으면 아무리 빨라도 3월 9일 2시 1분부터 사용가능)

# 예약이 시작되기 1시간전까지 또는 1시간 미만으로 남았을 경우 (59-현재 분)/2 만큼 코드가 멈춰있게 되어있음. 시간이 1시간 미만으로 남거나 (59-현재 분)/2 만큼 남은 경우 코드가 돌아가기 시작함.

# 개인이 수정해야하는 부분 ID, PWD, wish date, reservation date, 마지막 부분(cell과 confirm)의 tr 안의 숫자
	* wish date는 버스를 예약하고싶은 날짜
	* reservation date는 wish date보다 2일 낮게, 예약을 실제로 하는 날짜
	* tr 안의 숫자는 예약을 하고 싶은 시간으로, 원하는 시간대의 순서가 몇번째인이 찾은다음 수정하면 됨.

# 국캠-신촌, 신촌-국캠 예약이 모두 포함되어 있고, 경쟁이 심한 국캠-신촌이 먼저 예약되도록 세팅하였음. 원하는 경우 하나를 예외처리하고 사용하여도 무관.
